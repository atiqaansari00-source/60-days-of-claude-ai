# Day 2 - What is Prompt Engineering

## What I Learned
Today I learned about Prompt Engineering and how writing better prompts helps AI give more accurate and useful responses. I understood that structured prompts with context, role, and clear instructions improve output quality.

## Weak Prompt
**Prompt:**
Create an image explaining Prompt Engineering

**Output:**
The result was basic and generic with less structure.

## Engineered Prompt
**Prompt:**
You are an AI educator teaching complete beginners.

Explain Prompt Engineering in simple language.

Include:
- What Prompt Engineering is
- Why it matters when using AI tools like Claude
- One example of a weak prompt
- One example of an improved prompt
- Three practical benefits of writing better prompts

Also create a LinkedIn-ready image concept.

**Output:**
The response was more detailed, professional, structured, and visually better.

## Key Learning
Better prompts give better results. Adding context, format, and instructions makes AI responses more useful and accurate.
